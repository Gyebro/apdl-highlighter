MSHMID

MSHMID, KEY
Specifies placement of midside nodes.

PREP7: Meshing
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | DYNA

KEY

Key indicating placement of midside nodes:

0

 — 

Midside nodes (if any) of elements on a region boundary follow the curvature of the boundary line or area (the default).

1

 — 

Place midside nodes of all elements so that element edges are straight. Allows coarse mesh along curves.

2

 — 

Do not create midside nodes (elements will have removed midside nodes).

Notes

The MSHMID, MSHAPE, and MSHKEY commands replace the functionality that was provided by the ESHAPE command in ANSYS 5.3 and earlier releases.

This command is also valid for rezoning.
