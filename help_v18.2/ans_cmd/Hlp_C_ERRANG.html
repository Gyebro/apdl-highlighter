ERRANG

ERRANG, EMIN, EMAX, EINC
Specifies the element range to be read from a file.

PREP7: Elements
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | DYNA

EMIN, EMAX, EINC

Elements with numbers from EMIN (defaults to 1) to EMAX (defaults to 999999999) in steps of EINC (defaults to 1) will be read.

Notes

Defines the element number range to be read [EREAD] from the element file. If a range is also implied from the NRRANG command, only those elements satisfying both ranges will be read.

Menu Paths

Main Menu>Preprocessor>Modeling>Create>Elements>Read Elem File
