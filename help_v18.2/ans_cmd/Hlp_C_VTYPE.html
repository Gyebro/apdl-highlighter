VTYPE

VTYPE, NOHID, NZONE
Specifies the viewing procedure used to determine the form factors for the Radiation Matrix method.

AUX12: Radiation Matrix Method
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | –

NOHID

Type of viewing procedure:

0  

 — 

Hidden procedure.

1  

 — 

Non-hidden (faster, but less general) procedure.

NZONE

Number of sampling zones for the hidden procedure (100 maximum for 3-D, 1000 maximum for 2-D). Defaults to 20 for 3-D, 200 for 2-D. Number of points is 2*NZONE for 2-D and 2*NZONE*(NZONE+1) for 3-D.

Command Default

Hidden procedure using 20 zones for 3-D, 200 zones for 2-D.

Menu Paths

Main Menu>Radiation Opt>Matrix Method>Write Matrix
