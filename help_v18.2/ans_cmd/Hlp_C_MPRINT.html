MPRINT

MPRINT, KEY
Specifies that radiation matrices are to be printed.

AUX12: Radiation Matrix Method
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | –

KEY

Print key:

0  

 — 

Do not print matrices.

1  

 — 

Print matrices.

Command Default

Matrices are not printed.

Notes

Specifies that the element and node radiation matrices are to be printed when the WRITE command is issued. If KEY = 1, form factor information for each element will also be printed.

Menu Paths

Main Menu>Radiation Opt>Matrix Method>Write Matrix
