BETAD

BETAD, VALUE
Defines the stiffness matrix multiplier for damping.

SOLUTION: Dynamic Options
Compatible Products: – | – | Premium | Enterprise | Ent PP | Ent Solver | DYNA

VALUE

Stiffness matrix multiplier for damping.

Notes

This command defines the stiffness matrix multiplier used to form the viscous damping matrix , where is the stiffness matrix.

Values of can also be input as a material property (MP,BETD or TB,SDAMP,,,,BETD). If BETD in either form is included, the BETD value is added to the BETAD value as appropriate. (See Material Damping in the Mechanical APDL Material Reference and Damping Matrices in the Mechanical APDL Theory Reference.) Damping is not used in static (ANTYPE,STATIC) or buckling (ANTYPE,BUCKLE) analyses.

This command is also valid in PREP7.
