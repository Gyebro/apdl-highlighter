RMRGENERATE

RMRGENERATE
Performs fitting procedure for all ROM functions to generate response surfaces.

REDUCED ORDER MODELING: Generation Pass
Compatible Products: – | – | – | Enterprise | Ent PP | Ent Solver | –

Notes

The fitting procedure uses modal analysis data and function data generated using the RMSMPLE command and specifications set forth in the RMROPTIONS command. The files jobname_ijk.pcs (modes i, j, k) will be generated containing the coefficients of the response surfaces. These files are needed for the ROM Use Pass along with a ROM data base file [RMSAVE].

Distributed ANSYS Restriction This command is not supported in Distributed ANSYS.

Input Files

Strain energy and capacitance data file jobname_ijk.dec

Output Files

Response surface coefficients jobname_ijk.pcs (modes i, j, k)

Menu Paths

Main Menu>ROM Tool>Resp Surface>Fit Functions
