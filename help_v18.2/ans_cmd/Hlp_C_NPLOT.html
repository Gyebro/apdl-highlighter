NPLOT

NPLOT, KNUM
Displays nodes.

PREP7: Nodes
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | DYNA

KNUM

Node number key:

0  

 — 

No node numbers on display.

1  

 — 

Include node numbers on display. See also /PNUM command.

Notes

Produces a node display. Only selected nodes [NSEL] are displayed. Elements need not be defined. See the DSYS command for display coordinate system.

This command is valid in any processor.
