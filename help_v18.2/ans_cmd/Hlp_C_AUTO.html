/AUTO

/AUTO, WN
Resets the focus and distance specifications to "automatically calculated."

GRAPHICS: Views
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | DYNA

WN

Window number (or ALL) to which command applies (defaults to 1).

Notes

Focus point and distance will be automatically calculated during next display. Settings may still be changed with the /FOCUS and /DIST commands after this command has been issued. See also the /USER command.

This command is valid in any processor.

Menu Paths

Utility Menu>PlotCtrls>Pan, Zoom, Rotate
Utility Menu>PlotCtrls>View Settings>Automatic Fit Mode
