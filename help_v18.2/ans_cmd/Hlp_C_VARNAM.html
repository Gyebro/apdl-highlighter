VARNAM

VARNAM, IR, Name
Names (or renames) a variable.

POST26: Set Up
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | DYNA

IR

Reference number of the variable (2 to NV [NUMVAR]).

Name

Thirty-two character name for identifying variable on printouts and displays. Embedded blanks are compressed for output.
