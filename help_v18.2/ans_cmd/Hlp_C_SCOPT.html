SCOPT

SCOPT, TempDepKey
Specifies System Coupling options.

SOLUTION: Analysis Options
Compatible Products: Pro | Premium | Enterprise | Ent PP | Ent Solver | –

TempDepKey

Temperature-dependent behavior key based on the convection coefficient:

YES

 — 

A negative convection coefficient, -N, is assumed to be a function of temperature and is determined from the HF property table for material N (MP command). This is the default.

NO

 — 

A negative convection coefficient, -N, is used as is in the convection calculation.

Notes

By default in the Mechanical APDL program, a negative convection coefficient value triggers temperature-dependent behavior. In System Coupling, and in some one-way CFD to Mechanical APDL thermal simulations, it is desirable to allow convection coefficients to be used as negative values. To do so, issue the command SCOPT,NO.

Menu Paths

This command cannot be accessed from a menu.
