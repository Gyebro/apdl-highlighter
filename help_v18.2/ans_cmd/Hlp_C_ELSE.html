*ELSE

*ELSE
Separates the final if-then-else block.

APDL: Process Controls
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | DYNA

Notes

Optional final block separator within an if-then-else construct. See the *IF for details. If a batch input stream hits an end-of-file during a false *IF condition, the ANSYS run will not terminate normally. You will need to terminate it externally (use either the Linux “kill” function or the Windows task manager). The *ELSE command must appear on the same file as the *IF command, and all five characters must be input.

This command is valid in any processor.

Menu Paths

This command cannot be accessed from a menu.
