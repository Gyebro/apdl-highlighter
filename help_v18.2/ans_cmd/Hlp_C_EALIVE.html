EALIVE

EALIVE, ELEM
Reactivates an element (for the birth and death capability).

SOLUTION: Birth and Death
Compatible Products: – | – | – | Enterprise | Ent PP | Ent Solver | –

ELEM

Element to be reactivated. If ALL, reactivate all selected elements [ESEL]. If ELEM = P, graphical picking is enabled and all remaining command fields are ignored (valid only in the GUI). A component name may also be substituted for ELEM. To specify a table, enclose the table name in percent signs (%), e.g. EALIVE,%tabname%.

Notes

Reactivates the specified element when the birth and death capability is being used. An element can be reactivated only after it has been deactivated (EKILL).

Reactivated elements have a zero strain (or thermal heat storage, etc.) state.

The usage of tabular input is described in Using TABLE Type Array Parameters to Control Birth and Death in the Advanced Analysis Guide.

Do not use element deactivation/reactivation in analyses involving time-dependent materials, such as viscoelasticity, viscoplasticity, and creep analysis.

This command is also valid in PREP7.
