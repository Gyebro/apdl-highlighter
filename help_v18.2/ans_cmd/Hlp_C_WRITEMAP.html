WRITEMAP

WRITEMAP, Fname
Writes interpolated pressure data to a file.

/MAP: Pressure Mapping
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | –

Fname

File name and directory path (248 characters maximum, including the characters needed for the directory path). An unspecified directory path defaults to the working directory; in this case, you can use all 248 characters for the file name.

Notes

Writes the interpolated pressure data to the specified file. The data is written as SFE commands applied to the SURF154 elements that are on the target surface. You may read this data for inclusion in an analysis by using /INPUT,Fname.

Menu Paths

This command cannot be accessed from a menu.
