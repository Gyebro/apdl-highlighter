*LSFACTOR

*LSFACTOR, EngineName, Option
Performs the numerical factorization of a linear solver system.

APDL: Matrix Operations
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | DYNA

Argument Descriptions

EngineName

Name used to identify this engine. Must have been previously created using *LSENGINE.

Option

Option to invert the matrix, used only with an LAPACK engine (*LSENGINE,LAPACK):

INVERT -- 

Invert the matrix.

Notes

Performs the computationally intensive, memory intensive factorization of a matrix specified by *LSENGINE, using the solver engine also specified by *LSENGINE.

Menu Paths

This command cannot be accessed from a menu.
