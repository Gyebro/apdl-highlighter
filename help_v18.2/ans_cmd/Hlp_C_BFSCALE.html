BFSCALE

BFSCALE, Lab, FACT, TBASE
Scales body force loads at nodes.

SOLUTION: FE Body Loads
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | –

Lab

Valid body load label. If ALL, use all appropriate labels.

FACT

Scale factor for the nodal body load values. Zero (or blank) defaults to 1.0. Use a small number for a zero scale factor. The scale factor is not applied to body load phase angles.

TBASE

Base temperature for temperature difference. Used only with Lab = TEMP. Scale factor is applied to the temperature difference (T - TBASE) and then added to TBASE. T is the current temperature.

Notes

Scales body force loads in the database on the selected nodes. Issue the BFLIST command to list the nodal body loads. Solid model boundary conditions are not scaled by this command, but boundary conditions on the FE model are scaled.

Note:  Such scaled FE boundary conditions may still be overwritten by unscaled solid model boundary conditions if a subsequent boundary condition transfer occurs.

BFSCALE does not work for tabular boundary conditions.

This command is also valid in PREP7.
