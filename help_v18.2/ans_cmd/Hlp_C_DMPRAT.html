DMPRAT

DMPRAT, RATIO
Sets a constant modal damping ratio.

SOLUTION: Dynamic Options
Compatible Products: – | – | Premium | Enterprise | Ent PP | Ent Solver | DYNA

RATIO

Modal damping ratio (for example, 2% is input as 0.02).

Command Default

Use damping as defined by Damping in the Structural Analysis Guide.

Notes

Sets a constant damping ratio for use in the mode-superposition transient (ANTYPE,TRANS) or harmonic (ANTYPE,HARMIC) analysis and the spectrum (ANTYPE,SPECTR) analysis.

This command is also valid in PREP7.

Additional Information

Damping Matrices in the Mechanical APDL Theory Reference.
