RMNEVEC

RMNEVEC
Extracts neutral plane eigenvectors from a modal analysis for the ROM method.

REDUCED ORDER MODELING: Preparation
Compatible Products: – | – | – | Enterprise | Ent PP | Ent Solver | –

Notes

This command extracts the eigenvectors at a neutral plane of a model from a modal analysis. The modal analysis must have expanded modes [MXPAND] in order to process the data. Only the first 9 modes are considered. The command requires a node component named "NEUN" to be defined. These nodes represent the nodes at the neutral plane of a structure (in the case of a stress-stiffened structure), or at any plane in the structure (non stress-stiffened case).

This command is only valid in POST1.

Distributed ANSYS Restriction This command is not supported in Distributed ANSYS.

Output Files

jobname.evx, jobname.evy, jobname.evz, jobname.evn, jobname.evl

Menu Paths
