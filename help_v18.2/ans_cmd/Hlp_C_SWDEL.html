SWDEL

SWDEL, Ecomp
Deletes spot weld sets.

PREP7: Elements
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | –

Ecomp

Name of an existing spot weld set that was previously defined using SWGEN. If Ecomp = ALL (default) all spot welds are deleted.

Notes

This command deletes spot weld sets previously defined by the SWGEN command.

Menu Paths

Main Menu>Preprocessor>Modeling>Create>Elements>SpotWeld>Delete
