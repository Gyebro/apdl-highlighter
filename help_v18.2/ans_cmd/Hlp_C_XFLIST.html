XFLIST

XFLIST, EnrichmentID
Lists enrichment details and associated crack information

PREP7: Special Purpose
SOLUTION: Misc Loads
Compatible Products: – | – | – | Enterprise | Ent PP | Ent Solver | –

EnrichmentID or (blank)

Name of the enrichment specified via the associated XFENRICH command. Specifying EnrichmentID is optional.

If no value (or ALL) is specified, the command lists all enrichments and associated data.

Notes

This command is valid in PREP7 (/PREP7) and SOLUTION (/SOLU).

Menu Paths

This command cannot be accessed from a menu.
