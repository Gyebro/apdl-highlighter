ALPHAD

ALPHAD, VALUE
Defines the mass matrix multiplier for damping.

SOLUTION: Dynamic Options
Compatible Products: – | – | Premium | Enterprise | Ent PP | Ent Solver | DYNA

VALUE

Mass matrix multiplier for damping.

Notes

This command defines the mass matrix multiplier used to form the viscous damping matrix , where is the mass matrix.

Values of can also be input as a material property (MP,ALPD or TB,SDAMP,,,,ALPD). If ALPD in either form is included, the ALPD value is added to the ALPHAD value as appropriate. (See Material Damping in the Mechanical APDL Material Reference and Damping Matrices in the Mechanical APDL Theory Reference.) Damping is not used in static (ANTYPE,STATIC) or buckling (ANTYPE,BUCKLE) analyses.

This command is also valid in PREP7.
