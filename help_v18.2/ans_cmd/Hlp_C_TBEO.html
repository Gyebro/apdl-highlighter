TBEO

TBEO, Par, Value
Sets special options or parameters for material data tables.

PREP7: Data Tables
Compatible Products: – | – | – | Enterprise | Ent PP | Ent Solver | –

Par

Parameter name:

CAPCREEPREG

 — 

Available for the viscoplasticity/creep model (TB,CREEP), allows two creep models to be specified via the same material ID when used with the Extended Drucker-Prager model (TB,EDP).

FDCS

 — 

Coordinate system to use with location (XCOR, YCOR, ZCOR) or displacement (UX, UY, UZ) field variables.

Value

Parameter value:

When Par = CAPCREEPREG --

SHEA

 — 

Use the shear stress-state creep model with the Extended Drucker-Prager model.

COMP

 — 

Use the compaction stress-state creep model with the Extended Drucker-Prager model.

When Par = FDCS --

Any predefined, user-defined, or custom (LOCAL or CS) coordinate system number.

Notes

Issue the TBEO command after activating the material data table (TB) but before defining data for the table (TBDATA) or a point on a nonlinear data curve (TBPT).

If the defined material data table has subtables, issue the TBEO command for each desired subtable.

Menu Paths

This command cannot be accessed from a menu.
