MRPM

MRPM, VAL1
Defines the revolutions per minute (RPM) for a machine rotation.

SOLUTION: Misc Loads
Compatible Products: – | – | – | Enterprise | Ent PP | Ent Solver | –

VAL1

The RPM value (no default).

Notes

MRPM defines an RPM value for use in an acoustic analysis. A different RPM value can be defined at each load step. The RPM value is used to postprocess the equivalent radiated power from the structural surface (the PRAS and PLAS commands) or the radiated sound power level (the PRFAR and PLFAR commands).

Menu Paths

This command cannot be accessed from a menu.
