CZDEL

CZDEL, grp1, grp2, grp3
Edits or clears cohesive zone sections.

PREP7: Meshing
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | –

grp1

Initial group of cohesive zone elements to be deleted.

grp2

Final group of cohesive zone elements to be deleted.

grp3

Increment for selected groups.

Notes

The CZDEL command edits or deletes the interface elements and nodes, along with the associated changes made to the underlying plane or solid elements created during a previous CZMESH operation.

Each CZMESH operation will create groups of elements and nodes with component names in the format CZME_EL01 (elements) and CZME_ND01 (nodes). The final number of this format will be the number used for grp1 and grp2. If grp1 = ALL, all nodes and elements created by the CZMESH command will be deleted. After using CZDEL, all the user-defined components will be unselected.

The CZDEL command is valid for structural analyses only.

Menu Paths

This command cannot be accessed from a menu.
