LIST

LIST, LEVEL
Lists out the sets in the results file.

AUX3: Results Files
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | DYNA

Notes

This command lists the results set number, the load step, substep, and time step for each set. It also shows all sets marked for deletion.

Menu Paths

This command cannot be accessed from a menu.
