GMFACE

GMFACE, Lab, N
Specifies the facet representation used to form solid models.

GRAPHICS: Style
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | DYNA

Lab

Valid Labels:

FINE  

 — 

Value that determines how coarse the facets will be.

N

An integer value between one (small) and ten (large) that determines the tolerances that will be applied to the creation of arcs and surfaces. Ten will create many facets, which may in turn cause ANSYS to run very slowly. One will create fewer facets, which may in turn cause larger tolerance errors.
