EDCPU

EDCPU, CPUTIME
Specifies CPU time limit for an explicit dynamics analysis.

SOLUTION: Explicit Dynamics
Compatible Products: – | – | – | – | – | – | DYNA

CPUTIME

CPU time limit (in seconds) for the current phase of the analysis (defaults to 0). If CPUTIME = 0, no CPU time limit is set. CPUTIME values below 0 are not allowed.

Notes

This command is also valid in PREP7.

Distributed ANSYS Restriction This command is not supported in Distributed ANSYS.

Menu Paths

Main Menu>Solution>Analysis Options>CPU Limit
