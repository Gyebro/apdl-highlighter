GAPF

GAPF, NVAR, NUM, Name
Defines the gap force data to be stored in a variable.

POST26: Set Up
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | –

NVAR

Arbitrary reference number assigned to this variable (2 to NV [NUMVAR]). Overwrites any existing results for this variable.

NUM

Number identifying gap number for which the gap force is to be stored. Issue the GPLIST command to display gap numbers.

Name

Thirty-two character name for identifying the item on the printout and displays (defaults to the name GAPF).

Notes

Defines the gap force data to be stored in a variable. Applicable only to the expansion pass of the mode-superposition linear transient dynamic (ANTYPE,TRANS) analysis. The data is usually on Fname.RDSP.
