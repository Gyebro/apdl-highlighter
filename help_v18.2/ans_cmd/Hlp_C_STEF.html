STEF

STEF, VALUE
Specifies Stefan-Boltzmann radiation constant.

AUX12: General Radiation
SOLUTION: Radiosity
Compatible Products: – | Pro | Premium | Enterprise | Ent PP | Ent Solver | –

VALUE

Stefan-Boltzmann constant (defaults to 0.119E-10 Btu/hr/in2/ °R4).

Command Default

0.119E-10 Btu/hr/in2/ °R4.

Notes

You can use this command in the general preprocessor (PREP7) and in the Solution processor to specify the Stefan-Boltzmann constant in analyses using the radiation matrix method or the radiosity solver to model radiation.
