/ICLWID

/ICLWID, FACTOR
Scales the line width of circuit builder icons.

GRAPHICS: Scaling
Compatible Products: – | – | – | Enterprise | Ent PP | Ent Solver | –

FACTOR

Multiplication factor applied to the default line width (defaults to 1). The minimum is 1 and the maximum is 6.

Notes

Distributed ANSYS Restriction This command is not supported in Distributed ANSYS.

Menu Paths

Main Menu>Preprocessor>Modeling>Create>Circuit>Scale Icon
