6.5. Elements for Diffusion Analysis

The elements listed in this section are available for a diffusion analysis.

Use the following diffusion elements for 2-D simulation:

  • PLANE238 - 2-D 8-Node Diffusion Solid

Use the following diffusion elements for 3-D simulation:

  • SOLID239 - 3-D 20-Node Diffusion Solid

  • SOLID240 - 3-D 10-Node Tetrahedral Diffusion Solid

For more information, see Diffusion in the Mechanical APDL Theory Reference.
