FLUID221 Element Description

FLUID221 is a higher order 3-D 10-node solid element that exhibits quadratic pressure behavior. Use it for modeling the fluid medium and the interface in fluid-structure interaction problems.

For detailed information about this element, see Elements for Acoustic Analysis.
