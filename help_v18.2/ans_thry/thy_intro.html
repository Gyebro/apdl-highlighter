Chapter 1: Introduction

The Mechanical APDL Theory Reference presents theoretical descriptions of many elements, commands, and procedures in Mechanical APDL and related products. It is available to any customer who wants to better understand how the program uses input data to calculate output, and how to interpret various element and command results.

This reference is a legacy document no longer updated by ANSYS, Inc. Instead, new theoretical information appears in the respective analysis guides, and ANSYS, Inc. continues to migrate current theoretical information to the guides. Some content in this reference may still be useful to Mechanical APDL users, however, so ANSYS, Inc. continues to provide the document.
