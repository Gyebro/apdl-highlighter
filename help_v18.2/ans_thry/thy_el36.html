13.36. SOURC36 - Current Source

13.36.1. Description

The functionality of SOURC36 is basically one of user convenience. It provides a means of specifying the necessary data to evaluate the Biot-Savart integral (Equation 5–18) for the simple current source configurations, coil, bar and arc. The magnetic field {Hs} that results from this evaluation in turn becomes a load for the magnetic scalar potential elements (SOLID5, SOLID96 and SOLID98) as discussed in Electromagnetics.
