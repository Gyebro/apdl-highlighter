13.279. SOLID279 - 3-D 20-Node Homogeneous/Layered Thermal Solid

SOLID279 is available in two forms:

  • Homogeneous (nonlayered) thermal solid (KEYOPT(3) = 0, the default) - see SOLID279 - 3-D 20-Node Homogeneous Thermal Solid.

  • Layered thermal solid (KEYOPT(3) > 0) - see SOLID279 - 3-D 20-Node Layered Thermal Solid.
